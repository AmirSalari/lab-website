Our Projects

we are primarily interested in developing a framework for algorithm design in security as well as in studying security protocols from communication, coding and information theoretic view points.

Intrusion Detection System

An intrusion detection system or IDS is a device or software application that monitors a network or systems for malicious activity or policy violations. Any intrusion activity or violation is typically reported either to an administrator or collected centrally using a security information and event management system. Here we worked on Network Intrusion Detection System.

Congestion Free Router Networking

As a result of its strict adherence to end-to-end congestion control, the current Internet suffers from two maladies: Congestion collapse from undelivered packets, and unfair allocations of bandwidth between competing traffic flows. we introduce a novel Internet traffic control protocol called Congestion Free Router.

Learn more

P2P resource monitoring system

We present a peer-to-peer resource monitoring system for general purpose Unix LAN systems. It is designed to monitor system resources on a single LAN, but it also could be deployed on several LANs. Its peer-to-peer design makes it a scalable and fault tolerant monitoring system for collecting resource usage information.
